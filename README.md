# Iron Web Server

 - Build the server :

```sh
cargo build --release
```

- Run the server : 

```sh
cd /target/release
./rust_server
```

## Plain text response
- Navigate to localhost:3000

## Json response
- Navigate to localhost:3000/YOUR-QUERY
